## Project Setup

## Step by Step Floww

### package.json File Create

```cmd
npm init -y

```

### install Express

```cmd
npm install express --save
```

### install Mongoose

```cmd
npm install mongoose --save
```

### install typescript devdepence

```cmd
npm install typescript --save-dev
```

### install Cors

```cmd
 npm i cors
```

### install corse type

```cmd
npm install --save @types/cors

```

### install Dotenv

```cmd
npm install dotenv --save
```
